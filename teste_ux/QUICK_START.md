🎯 INSTRUÇÕES PARA TESTES UX
============================

## 🚀 COMO INICIAR

1. Abra o terminal
2. Navegue para o diretório do projeto
3. Execute: `uv run ux_test_selector.py`
4. Escolha uma opção do menu

## 📋 OPÇÕES DISPONÍVEIS

- **1**: 🤖 Teste Service Agent (mais conversacional)
- **2**: 📋 Teste Workflow (mais estruturado)  
- **3**: 📚 Ver guia de teste completo
- **0**: Sair

## 🧪 CENÁRIOS ESSENCIAIS

### Teste Básico
```
"Oi, quero me identificar. Meu CPF é 111.444.777-35"
```

### Teste Validação
```
"Meu CPF é 111.444.777-99"
```

### Teste Serviços
```
"Preciso de ajuda com IPTU"
```

## 📊 O QUE AVALIAR

- ⏱️ Tempo de resposta
- 💬 Qualidade da conversa
- 🎯 Facilidade de uso
- ✅ Validações funcionam

## 🎉 DECISÃO

Testem ambos e escolham o que oferece melhor experiência para os cidadãos!
